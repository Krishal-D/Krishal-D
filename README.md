## Featured Projects

### [TalentTrack — Applicant Tracking System](https://github.com/Krishal-D/ats-project) &nbsp; [![Live](https://img.shields.io/badge/live-kd--talenttrack.vercel.app-10b981)](https://kd-talenttrack.vercel.app)

Production-style hiring platform with end-to-end full-stack architecture deployed across three services.

- React 19 SPA (Vercel) → Express 5 API (Render) → PostgreSQL (Supabase)
- JWT access + refresh token rotation with HttpOnly cookies
- Role-based authorization (candidate vs employer — separate UIs and endpoints)
- Resume upload with MIME type and size validation (Multer)
- Normalised database schema with foreign keys, cascading deletes, and joined queries
- Application pipeline: pending → reviewed → shortlisted → rejected
- Profile management with photo upload, responsive nav with hamburger menu
