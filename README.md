## 🚀 What I Build

I build and deploy full-stack web applications from scratch — database schema through to cloud deployment.

My flagship project **TalentTrack** is a production-style Applicant Tracking System featuring:
- JWT authentication with refresh token rotation (HttpOnly cookies)
- Role-based access control (candidate + employer personas)
- File upload handling with MIME validation
- Normalised PostgreSQL schema with cascading deletes
- Deployed across Vercel (frontend), Render (backend), and Supabase (database)
- Containerised with Docker Compose for local development

🔗 [Live Demo](https://kd-talenttrack.vercel.app) · [Source Code](https://github.com/Krishal-D/ats-project)
